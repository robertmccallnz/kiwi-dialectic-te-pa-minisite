# تي پا توواتاواتا — دليل تأهيل الشركاء

> **سيادة البيانات الأصلية · تعليم الذكاء الاصطناعي · الوصول إلى واجهة برمجة التطبيقات** (right-to-left)

---

أهلاً بك في شبكة شركاء تي پا توواتاواتا. يمنحك هذا الدليل كل ما تحتاجه للوصول إلى مكتبة الزخارف والحقائب التعليمية وواجهة برمجة التطبيقات العامة — ودمجها في منصاتك وإصداراتك أو برامجك التعليمية.

---

## عن تي پا توواتاواتا

تي پا توواتاواتا هي مؤسسة خيرية مقرها في بورت تشالمرز، أوتياروا نيوزيلندا. نبني بنية تحتية تعليمية مفتوحة لسيادة البيانات الأصلية ومحو أمية الذكاء الاصطناعي وأنظمة المعرفة التحررية — متجذرة في تقاليد الزخارف الثقافية لشعوب ماوري وساموا والمحيط الهادئ والغواراني والشيبيبو وغونا وكايابو ويانومامي وفيجي وتونغا والأمازون.

---

## ما يحصل عليه الشركاء

- الوصول إلى 30 زخرفة ثقافية أصلية — صور PNG وملفات متجهة SVG وصور ميم ناشطة
- حقائب تعليمية بـ 6 لغات (الإنجليزية، التي ريو ماوري، البرتغالية، أڤانييي، الساموية، العربية)
- واجهة برمجة تطبيقات عامة للقراءة فقط — 100 طلب مجاني/ساعة، لا يلزم مفتاح
- مجموعة وسائل التواصل الاجتماعي — 624 بطاقة SVG محسّنة لفيسبوك وإنستغرام وتيكتوك وX
- صور ميم PNG خاصة باللغة لجميع الزخارف الـ 30 × 6 لغات
- رخصة CC BY-NC-SA 4.0 — مجانية الاستخدام والتكيف مع الإسناد

---

## واجهة برمجة التطبيقات العامة

لا تتطلب واجهة البرمجة للقراءة فقط أي مصادقة. جميع نقاط النهاية تُرجع JSON مع إسناد كامل مدمج.

**عنوان URL الأساسي:** `https://te-pa-analytics.sketchschool.workers.dev`

| Endpoint | الوصف | Example |
|---|---|---|
| `GET /api/motifs` | جميع الزخارف الـ 30 مع المعاني وعناوين CDN وروابط الميم | `https://te-pa-analytics.sketchschool.workers.dev/api/motifs?lang=ar` |
| `GET /api/motifs/:id` | زخرفة واحدة حسب المعرّف — قائمة كاملة بالأصول | `https://te-pa-analytics.sketchschool.workers.dev/api/motifs/koru` |
| `GET /api/cultures` | قائمة بجميع الثقافات الـ 11 مع أعداد الزخارف | `https://te-pa-analytics.sketchschool.workers.dev/api/cultures` |
| `GET /api/meme` | عنوان CDN لصورة ميم PNG بلغة محددة | `https://te-pa-analytics.sketchschool.workers.dev/api/meme?id=koru&lang=ar` |
| `GET /api/kit` | عنوان URL لملف PDF للحقيبة التعليمية بلغة محددة | `https://te-pa-analytics.sketchschool.workers.dev/api/kit?lang=ar` |

> يُرجى تضمين الترويسة X-TePa-Use: <اسم مشروعك> في طلباتك. يساعدنا ذلك على فهم كيفية استخدام الواجهة ودعم طلبات التمويل المستقبلية.

## التكامل مع وسائل التواصل الاجتماعي

كل زخرفة لها مجموعة اجتماعية كاملة مع بطاقات محسّنة لكل منصة. الوسوم الموصى بها لمنشوراتك:

```
#سيادة_البيانات  #الذكاء_الاصطناعي_الأصلي  #TePa  #CARE  #حقوق_البيانات  #الشعوب_الأصلية
```

عند النشر، أشر إلى @KiwiDialectic وأضف رابطاً إلى te-pa.org/rhizome-mapper.html ليتمكن جمهورك من استكشاف الشبكة الكاملة.

---

## البدء السريع — مثال كود

```javascript
// جلب جميع الزخارف باللغة العربية
fetch('https://te-pa-analytics.sketchschool.workers.dev/api/motifs?lang=ar', {
  headers: { 'X-TePa-Use': 'اسم-مشروعك' }
})
.then(r => r.json())
.then(data => {
  data.motifs.forEach(m => {
    console.log(m.name_en, m.assets.meme_png);
  });
});
```
