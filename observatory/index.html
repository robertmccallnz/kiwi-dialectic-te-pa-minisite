Te Pā Collective Action Lab

Systems Observatory

Twelve leverage points · Aotearoa political economy · a living archive

A living dashboard of Donella Meadows' twelve leverage points — the places where a system can be intervened in, from the shallowest tweak of a parameter down to the deepest shift of the paradigm it arises from. Each of the twelve panels maps a Meadows tier to a public Aotearoa indicator, watches it against its own 24-period baseline, and flags deviations that challenge the current political-economic paradigm.

He aha tēnei — What this is

Meadows' framework, wired to Aotearoa data

Meadows argued that most policy energy is spent at the shallowest tiers — twiddling numbers, patching feedback loops — while the deepest leverage sits in the paradigm the system takes for granted. The Observatory makes that claim inspectable. It pairs each tier with a concrete NZ indicator, refreshes them weekly from public government sources, and lets kaimahi rank interventions by which tier they actually operate at.

Ngā rauemi — How it is built

Three transparent, forkable pieces

01 · Site

Static dashboard

GitHub Pages, versioned JSON, weekly refresh — the same pattern as the Thinkers Mapper. Nothing hidden; every number carries its source.

02 · Compute

Anomaly detection

A Hugging Face Space runs a rolling z-score against each indicator's own history. Deviations of |z| ≥ 2.5 are flagged as paradigm-challenging.

03 · Archive

Community annotation

Kaimahi rank interventions by leverage tier and cite whakapapa. Every submission is written to a public HF dataset — forkable, auditable, CC BY.

Kaupapa. Built under the Māori Data Governance Model of Te Kāhui Raraunga. Public, aggregated, non-personal data only. Every indicator cites its source. Te reo Māori terminology is used deliberately, as a paradigm-shifting act at leverage point #2.

Kōrero mai — Contribute

Rank an intervention. Cite a whakapapa.

The observatory is only as good as the readings kaimahi bring to it.
